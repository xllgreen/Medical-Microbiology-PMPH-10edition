# HDV感染类型评估与临床后果判断

## 适用对象
- HBV感染者
- HDV疑似病例
- 需进行肝炎重症风险评估的患者

## 前提条件
- 已确认患者存在HBV感染
- 患者具有HDV暴露风险

## 禁忌说明
- 不适用于未感染HBV的个体（因HDV为缺陷病毒，必须依赖HBV才能复制）

## 执行步骤

1. **区分HDV感染类型**：
   - **联合感染（co-infection）**：指既往未感染HBV者同时感染HBV和HDV。
   - **重叠感染（superinfection）**：指已有慢性HBV感染的患者继发HDV感染。

2. **评估临床后果差异**：
   - **联合感染**：通常表现为急性肝炎，部分病例可自限。
   - **重叠感染**：
     - 常导致肝炎显著加重；
     - 易进展为重型肝炎；
     - 加速肝硬化及肝细胞癌的发生。

3. **启动诊断流程**：
   - 检测血清抗-HD IgM（感染后2周出现，4～5周达峰）以识别早期感染；
   - 检测抗-HD IgG持续高效价以提示慢性HDV感染；
   - 确认肝组织HDAg或血清/组织中HDV RNA作为确诊依据。

4. **结合流行病学背景**：
   - 在中国，HBsAg阳性人群中HDV感染率为0%～32%，南方地区高于北方。

## 行动建议
对慢性乙肝患者出现病情突然恶化者，应立即检测抗-HD IgM/IgG及HDV RNA以明确是否为HDV重叠感染，并据此制定干预策略。
