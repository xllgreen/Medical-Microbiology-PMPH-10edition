# 布鲁菌波浪热形成机制解析

## 适用场景
- 解释“波浪热”热型成因
- 评估慢性化风险
- 指导长期抗菌治疗

## 执行步骤
1. **确认感染前提**：
   - 经皮肤/黏膜/消化道等途径侵入
   - 被巨噬细胞吞噬，转为胞内寄生
2. **追踪致病循环**：
   - 潜伏期1–4周 → 局部淋巴结繁殖 → 首次菌血症（发热2–3周）
   - 细菌侵入肝、脾、骨髓 → 发热消退（间歇期）
   - 再次繁殖入血 → 新一轮发热
3. **识别典型表现**：体温周期性升高与间歇，称为“波浪热”。
4. **预警慢性化**：因免疫逃逸，易转为慢性感染。

> **注意**：本技能不适用于非胞内寄生菌感染或无潜伏期的急性中毒。
