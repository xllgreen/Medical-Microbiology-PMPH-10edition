# 噬菌体溶原性转换分析

## 何时使用
- 用户研究细菌毒力获得机制（如白喉毒素、肉毒毒素产生）；
- 需解释细菌抗原结构或血清型变化原因；
- 已确认细菌为溶原性状态，并怀疑其表型由噬菌体基因编码；
- 排除毒性噬菌体或其他非溶原性机制后，聚焦温和噬菌体整合效应。

## 执行步骤
1. **确认前提条件**：
   - 细菌已被温和噬菌体感染；
   - 噬菌体基因组已整合为前噬菌体；
   - 噬菌体携带功能基因（如 tox 基因）。

2. **识别新获得性状**：
   - 检查细菌是否表达噬菌体编码的性状（如毒素、溶素、抗原变异）；
   - 对比亲本菌株与溶原化菌株的表型差异。

3. **验证可逆性**：
   - 若前噬菌体丢失，相关性状应消失，可作为判断依据。

4. **排除混淆概念**：
   - 区分溶原性转换与噬菌体免疫（后者仅提供对同源噬菌体再感染的抵抗力）；
   - 区分假溶原性（噬菌体DNA未整合且不复制）。

5. **输出结论**：
   - 明确指出细菌是否因溶原性转换获得特定性状，并说明其生物学或临床意义（如致病性增强）。

> 注意：本技能不适用于毒性噬菌体感染场景，且仅当前噬菌体携带功能性外源基因时适用。
