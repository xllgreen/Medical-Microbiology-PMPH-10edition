# 微生态失调的诱因与疾病关联分析

## 触发条件
当满足以下任一条件时启动本技能：
- 患者出现与菌群失衡相关的临床表现（如假膜性肠炎、细菌性阴道病、难治性腹泻等）
- 存在明确干扰因素：广谱抗生素使用、西化饮食、激素变化、免疫抑制或应激状态
- 患者属于高风险群体：感染性疾病患者、代谢病患者、肿瘤患者或免疫异常个体

## 执行步骤

### 1. 确认微生态失调定义
判断是否为微生物之间或微生物与宿主间平衡被打破，由生理性组合转为病理性组合的状态。

### 2. 识别主要诱因
- **抗生素使用**：尤其广谱抗生素，可导致敏感菌被杀灭，耐药菌（如艰难拟梭菌）过度生长，引发二重感染
- **饮食改变**：西化饮食常致芽胞杆菌门减少、拟杆菌门增加、产短链脂肪酸菌减少
- **激素波动**：如女性雌激素下降导致乳杆菌减少、阴道pH升高，诱发细菌性阴道病
- **免疫抑制或应激**：改变肠道或黏膜表面微生物构成

### 3. 分类关联疾病
根据临床表现，将可能疾病分为两大类并匹配典型模式：

#### A. 感染性疾病
- **消化系统**：假膜性肠炎（艰难拟梭菌、白念珠菌）、急慢性腹泻（原籍菌流失形成恶性循环）
- **口腔**：龋齿（变异链球菌+蔗糖→pH↓→脱钙）；牙周病（伴放线放线杆菌向牙龈卟啉单胞菌演替）
- **呼吸道**：口咽菌误吸导致医院获得性肺炎
- **生殖道**：细菌性阴道病（加德纳菌+厌氧菌）、念珠菌性阴道炎
- **皮肤**：金黄色葡萄球菌或真菌感染

#### B. 非感染性疾病
- **肥胖**：革兰阴性杆菌↑ → LPS释放 → TLR4激活 → 慢性炎症 → 代谢紊乱
- **糖尿病**（T1DM/T2DM）：双歧杆菌、阿克曼菌等有益菌减少，促炎菌增多
- **结直肠癌**：ETBF、pks+大肠杆菌、具核梭杆菌升高；产丁酸菌减少；可生成亚硝胺等致癌物
- **炎症性肠病**（IBD）：芽胞杆菌门↓、假单胞菌门↑ → 黏蛋白降解 → 肠屏障破坏
- **超敏反应/哮喘**：早期定植肺炎链球菌等促进IgE升高；约氏乳杆菌具保护作用
- **神经心理疾病**：孤独症患者常见拟杆菌门↑、芽胞杆菌门↓，提示肠-脑轴失调

### 4. 输出结果
生成结构化警报，包含：
- 主要干扰因素
- 疾病类型（感染性/非感染性）
- 关键菌群变化（升高的致病菌与降低的保护菌）

> 注意：本技能仅用于辅助病因推断，不替代实验室检测或临床诊断。
