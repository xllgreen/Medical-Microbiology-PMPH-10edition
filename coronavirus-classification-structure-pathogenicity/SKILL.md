# 冠状病毒分类、结构与致病特征评估

## 何时使用
- 患者出现发热、咳嗽、咽痛等呼吸道感染症状
- 需区分是否由冠状病毒引起
- 需判断是否为高致病性冠状病毒（如SARS-CoV、MERS-CoV、SARS-CoV-2）感染
- 临床需评估重症风险、传播潜力及防控措施

## 执行步骤

### 1. 确定病毒分类
- 查询ICTV 2023分类标准：
  - **α-冠状病毒属**：HCoV-229E、HCoV-NL63（人类）；主要感染哺乳动物
  - **β-冠状病毒属**：HCoV-OC43、HCoV-HKU1、SARS-CoV、MERS-CoV、SARS-CoV-2（人类）
  - **γ/δ-冠状病毒属**：主要感染禽类，不感染人类
- 若检测到人类冠状病毒，确认其所属（α或β）

### 2. 分析病毒结构特征
- 确认是否具备以下典型结构（需电镜或分子检测支持）：
  - 包膜球形病毒，直径80–140 nm，表面有“日冕”状刺突
  - 单股正链RNA基因组（27–32 kb）
  - 关键结构蛋白：S（刺突）、E（包膜）、M（膜）、N（核蛋白）；部分β属含HE蛋白

### 3. 判断致病强度
- **轻症型**（α属及部分β属：229E、NL63、OC43、HKU1）：
  - 表现为普通感冒，占感冒病因15%–30%
  - 免疫保护弱，可重复感染
- **重症型**（β属：SARS-CoV、MERS-CoV、SARS-CoV-2）：
  - 可致肺炎、急性呼吸窘迫综合征（ARDS）
  - 老年人及基础疾病患者病死率显著升高

### 4. 综合评估与输出
- 结合病毒属、已知人类受体（如ACE2、DPP4、APN）及临床表现：
  - 若为SARS-CoV、MERS-CoV或SARS-CoV-2，立即启动高致病性病原体防控流程
  - 若为普通感冒型冠状病毒，按常规上呼吸道感染处理
- 输出警示：明确是否需隔离、加强监测或上报公共卫生部门

> **注意**：本技能不适用于非冠状病毒引起的呼吸道感染。普通感冒型冠状病毒感染后免疫保护弱，可重复感染。
