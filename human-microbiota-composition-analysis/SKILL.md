# 人体微生物群构成特征分析

## 适用条件
- 仅适用于与外界相通的腔道或体表（如皮肤、口腔、呼吸道、胃肠道、泌尿生殖系统）
- 仅适用于正常生理状态
- 不适用于无菌组织或器官内部（如血液、脑脊液）

## 执行步骤
1. **确认目标解剖部位**：明确需分析的人体部位（如结肠、口腔黏膜、阴道等）。
2. **识别微生物三大类群**：
   - 原核生物（细菌为主，含少量古菌）
   - 真核生物（主要是真菌，占比约0.1%）
   - 病毒（主要是噬菌体）
3. **获取该部位的构成特征**：
   - **种类**：列出典型属/种
   - **丰度**：提供数量级（如每克或每毫升的对数数量）
   - **优势菌门/属**：指出占主导地位的分类单元
4. **返回结构化结果**：按部位输出微生物种类、数量级及优势菌群信息。

## 典型部位概要
- **皮肤**：凝固酶阴性葡萄球菌、棒状杆菌、丙酸杆菌
- **口腔**：甲型溶血性链球菌、韦荣球菌；成年期含600余种细菌
- **呼吸道**：链球菌、普雷沃菌为优势菌属；上呼吸道可定植金黄色葡萄球菌
- **胃肠道**：胃内<10³–10⁶个/mL（假单胞菌门等）；结肠达10¹¹个/g（芽胞杆菌门、拟杆菌门等五大门）
- **泌尿生殖道**：男性以葡萄球菌、埃希菌为主；女性以乳杆菌为主（维持低pH）

> 注：古菌在肠道以甲烷短杆菌、甲烷球形菌为主；口腔以甲烷杆菌为主。
