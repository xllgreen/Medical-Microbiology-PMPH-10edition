## Обзор Проекта

Этот проект систематически объединяет основные области медицинской микробиологии, включая бактериологию, вирусологию, микологию, иммунологию, дезинфекцию и стерилизацию, науку о микробиоме и контроль инфекций, охватывая **153 ключевых навыка** в 9 основных категориях.

**Целевая аудитория**: Студенты-медики, клиницисты, лабораторные технологи, работники общественного здравоохранения, команды инфекционного контроля

**Учебник**: People's Medical Publishing House *Medical Microbiology* 10-е издание

**⚠️Риск⚠️**: Этот набор навыков охватывает микробиологическую диагностику, противомикробную терапию, инфекционный контроль и лабораторные процедуры, которые могут быть неправильно использованы в качестве независимых медицинских рекомендаций.
