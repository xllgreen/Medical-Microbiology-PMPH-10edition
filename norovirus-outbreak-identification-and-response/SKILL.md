# Norovirus Outbreak Identification and Response

## 触发条件
当秋冬季在托幼机构、医院、学校等场所出现突发性恶心、呕吐、水样腹泻的聚集性病例时，考虑诺如病毒感染。

## 执行步骤

1. **确认病毒特性与传播方式**：
   - 无包膜+ssRNA病毒，抵抗力强，耐热（60℃ 30分钟仍有感染性）、耐醇；
   - 主要经粪-口传播，也可通过呕吐物气溶胶经空气传播；
   - 传染源包括病人和隐性感染者。

2. **识别临床与流行病学特征**：
   - 潜伏期24–48小时；
   - 症状：突发呕吐、腹痛、水样腹泻，持续1–3天，多为自限性；
   - 常见媒介：污染水源、未煮熟海鲜、凉菜等。

3. **启动检测与确认**：
   - 在发病后48–72小时内采集标本；
   - 首选RT-PCR检测；宏基因组测序可用于病原鉴定；
   - 注意：尚不能人工培养，无法通过细胞培养确诊。

4. **实施防控措施**：
   - **个人防护**：加强手卫生（肥皂+流动水，因醇类无效）；
   - **环境消毒**：使用含氯消毒剂（如次氯酸钠）或过氧化氢，避免使用醇类；
   - **病例管理**：隔离患者至症状消失后至少48小时；
   - **食品安全**：暂停供应冷食、生食，确保食物充分加热。

5. **输出结果**：
   启动感染控制响应，包括隔离、消毒、健康教育及疫情上报。
