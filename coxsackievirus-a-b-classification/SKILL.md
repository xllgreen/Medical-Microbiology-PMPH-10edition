# 柯萨奇病毒A/B组分类及致病特征

## 适用条件
- 已分离出疑似柯萨奇病毒株
- 具备乳鼠感染模型或细胞培养系统
- 不适用于埃可病毒或其他肠道病毒的直接分类

## 分类步骤

1. **观察乳鼠麻痹类型**：
   - 若引起**肌肉松弛型麻痹**（flaccid paralysis），归为**A组柯萨奇病毒**（CVA）
   - 若引起**肌肉痉挛型麻痹**（spastic paralysis），归为**B组柯萨奇病毒**（CVB）

2. **评估细胞培养增殖能力**：
   - CVA包含23个血清型（1–22及24），其中**A1、A19、A22无法在体外细胞中培养**
   - CVB包含6个血清型（1–6），**可在多种培养细胞中稳定增殖**

3. **结合血清型与致病特征确认分组**：
   - A组典型型别：CVA16（手足口病）、CVA24（急性出血性结膜炎）
   - B组典型型别：CVB1–5（心肌炎、流行性胸痛、新生儿全身性感染）

4. **处理例外情况**：
   - 对于无法在细胞中培养的CVA型别（如A1、A19、A22），必须依赖乳鼠模型完成病毒分离与鉴定

> **注意**：分类结果应明确返回病毒所属组别（A组或B组）及其可能的血清型范围。
