# 鼠疫耶尔森菌毒力因子分析

## 适用场景
- 实验室鉴定高毒力鼠疫耶尔森菌株
- 评估疑似鼠疫样本的生物安全等级
- 解释宿主免疫逃逸或组织损伤机制
- 判断是否需按甲类传染病处理

## 执行步骤

1. **确认前提条件**：
   - 菌株携带pMT1、pCD1、pPCP1三种毒力质粒
   - 培养或感染环境温度为28–30°C（利于毒力因子表达）

2. **识别关键毒力因子表达**：
   - 检测F1抗原（荚膜糖蛋白）
   - 检测V/W抗原共表达
   - 检测Yops（耶尔森菌外膜蛋白）分泌
   - 检测Pla蛋白活性
   - 观察内毒素相关症状（发热、休克、DIC）

3. **评估毒力与风险**：
   - 若F1、V/W抗原或Yops任一阳性，判定为高毒力菌株
   - 立即按《中华人民共和国传染病防治法》甲类传染病管理
   - 注意：鼠毒素对人类致病性尚不明确，不可单独作为致病依据

4. **注意事项**：
   - F1抗原不耐热（100°C 15分钟失活），加热样本后不得用于抗原检测
   - 鼠毒素仅在细菌自溶后释放，常规培养上清中可能检测不到

> **输出模板**：若检测到F1、V/W抗原或Yops表达，提示高毒力鼠疫耶尔森菌株，需按甲类传染病处理。
