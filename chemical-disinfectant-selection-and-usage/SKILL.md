# Chemical Disinfectant Selection and Usage

## 触发条件
当需对不耐热物品、环境表面、空气或人体外部（皮肤、黏膜）进行消毒时，依据风险等级和微生物特性选择合适的化学消毒剂。

## 执行步骤

1. **评估前提条件**：
   - 明确消毒对象的危险等级（高、中、低风险）；
   - 确认目标微生物种类（是否含芽胞、分枝杆菌、病毒等）；
   - 评估物品材质对消毒剂的耐受性（如金属是否耐腐蚀）。

2. **按效能等级选择消毒剂**：
   - **高效消毒剂**：用于杀灭包括芽胞在内的所有微生物，适用于高风险器械或环境（如内镜、污水）。
     - 含氯消毒剂（如次氯酸钠）：用于饮水、皮肤、表面、污水；注意腐蚀性。
     - 过氧化物类（如过氧乙酸、过氧化氢）：可用于空气熏蒸；注意刺激性和稳定性。
     - 醛类（如戊二醛）：用于精密器械；作用时间≥4小时。
     - 环氧乙烷：气体灭菌，适用于不耐热物品；需确保残留浓度≤1ppm。
   - **中效消毒剂**：不能杀芽胞，但可杀繁殖体、分枝杆菌、多数病毒，适用于皮肤、体温计等。
     - 含碘消毒剂（碘酊、碘伏）：注意着色与刺激性。
     - 醇类（70%~75%乙醇）：作用5~10分钟，用于皮肤和小件器械。
   - **低效消毒剂**：仅杀多数繁殖体，适用于低风险表面或物品。
     - 季铵盐类（如苯扎溴铵）：禁与肥皂等阴离子物质合用。
     - 氯己定（洗必泰）：同样禁与阴离子合用。
     - 高锰酸钾：用于皮肤、黏膜、食具等。

3. **应用通用原则**：
   - 仅限外用或环境使用，不可用于人体内部；
   - 使用“适度、适量”，避免过长时间接触；
   - 注意毒性、腐蚀性及环境污染；
   - 目标是“有效消毒”同时“保障健康与环境安全”。

4. **输出结果**：
   推荐具体的消毒剂种类、有效浓度、作用时间及关键注意事项。
