# 脊髓灰质炎病毒详细特性

## 病毒结构
- 直径24–30 nm，无包膜，二十面体对称。
- 单正链RNA基因组。
- 衣壳蛋白VP1–VP4，其中VP1为主要外露蛋白，是中和抗体的主要靶点。

## 受体与组织嗜性
- CD155（也称PVR，属于免疫球蛋白超家族）是唯一功能性受体。
- 表达于：脊髓前角运动神经元、背根神经节、骨骼肌、胎盘、某些免疫细胞。
- 决定宿主范围（仅人类和少数灵长类易感）。

## 全球流行现状（截至2023年）
- II型野毒株已于2015年宣布消灭；
- III型野毒株于2019年宣布消灭；
- I型野毒株仅在阿富汗和巴基斯坦局部传播；
- VDPV（疫苗衍生株）每年仍导致全球上千例VAPP，主要源于OPV中II型组分的回复突变，故多数国家已停用tOPV，改用bOPV（仅I、III型）。

## 疫苗比较
| 特性 | IPV（灭活疫苗） | OPV（口服减毒活疫苗） |
|------|------------------|------------------------|
| 接种方式 | 肌注 | 口服 |
| 抗体类型 | 血清IgG/IgM | sIgA + 血清抗体 |
| 群体免疫 | 无 | 有（通过接触传播间接免疫） |
| VAPP风险 | 无 | 有（约1/270万剂） |
| 对排毒抑制 | 弱 | 强 |

## 免疫保护机制
- **sIgA**：在咽喉和肠道黏膜阻断病毒吸附，减少排毒和传播；
- **血清IgG/IgM**：中和血流中病毒，防止其进入中枢神经系统；
- **母传抗体**：IgG经胎盘传递，可保护6月龄内婴儿，但可能干扰早期OPV效果。
