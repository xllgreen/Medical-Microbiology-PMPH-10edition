# 分析侵入性诊疗导致医院感染的机制

## 何时使用
- 患者即将或已经接受侵入性检查（如支气管镜、膀胱镜、胃镜）或治疗（如气管插管、留置导尿管、中心静脉置管、伤口引流、心导管、人工瓣膜植入）。
- 患者存在免疫低下风险（如器官移植、长期使用免疫抑制剂、严重基础疾病）。
- 需评估器械相关感染的潜在机制以制定预防或干预措施。

## 执行步骤
1. **识别屏障破坏类型**：
   - 检查操作是否破坏皮肤或黏膜物理屏障。
   - 确认是否将局部微生物群带入深层组织。

2. **评估器械污染风险**：
   - 核实所用器械是否经过彻底消毒灭菌。
   - 若消毒不彻底，判定可能引入外源性病原体。

3. **判断生物材料相关风险**：
   - 确认所用生物材料（如硅胶、聚氨酯导管）是否易被细菌黏附。
   - 评估留置时间是否延长，增加生物被膜形成概率。

4. **分析细菌生物被膜形成机制**：
   - 判断是否已出现以下过程：
     - 细菌分泌胞外多糖；
     - 细菌相互粘连形成生物被膜；
     - 生物被膜导致抗生素敏感性下降；
     - 生物被膜帮助细菌逃避免疫清除。

5. **输出风险提示**：
   - 提示该侵入性操作存在高感染风险；
   - 建议加强无菌操作、尽可能缩短器械留置时间、定期监测生物被膜形成迹象。

> **注意**：本技能不适用于非侵入性操作或皮肤/黏膜屏障完整未受损的情况。
