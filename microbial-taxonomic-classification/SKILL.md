# 微生物的三大分类体系

## 何时使用
- 需对未知微生物进行初步归类
- 需判断微生物是否具有细胞结构、核膜或特定核酸类型
- 在医学、环境或基础研究中识别微生物基本类别

## 执行步骤

1. **判断是否有细胞结构**：
   - 若无细胞结构 → 归为【非细胞型微生物】（如病毒）
     - 特征：无能量代谢酶系统，仅含一种核酸（DNA 或 RNA），必须在活细胞内寄生增殖
   - 若有细胞结构 → 进入第2步

2. **检查细胞核分化程度**：
   - 若细胞核高度分化（具核膜、核仁，细胞器完整）→ 归为【真核细胞型微生物】（如真菌）
   - 若为原始核质（裸露 DNA 团块，无核膜/核仁，仅有核糖体）→ 归为【原核细胞型微生物】

3. **对原核细胞型微生物进一步细分**：
   - **古菌**：无肽聚糖，具独特代谢途径，常见于极端环境（高温、高盐等），人体内存在但目前未发现致病性
   - **细菌**：通常具细胞壁（支原体除外），与人类健康密切相关；包括放线菌、螺旋体、衣原体、立克次体等特殊类型

## 关键变量
- `cell_structure`（布尔值）：是否有细胞结构
- `nuclear_differentiation`（枚举）：高（有核膜/核仁）或低（原始核质）
- `nucleic_acid_type`（字符串）：DNA、RNA 或两者兼有

## 输出结果
返回微生物所属类别（非细胞型、原核细胞型、真核细胞型）及关键特征说明。
