# 洋葱伯克霍尔德菌复合群的鉴定与耐药管理

## 何时使用
- 分离到革兰氏阴性需氧杆菌，且来源为：
  • 囊性纤维化患者呼吸道；
  • 医院水源、喷雾器、导尿管等污染源；
  • 免疫功能低下患者的血流或肺部感染；
- 表型特征可疑（如氧化酶阳性、产黄色色素但多数不产）。

## 执行步骤

### 1. 初步表型筛查
观察以下特征：
- 革兰氏阴性杆菌，具3～8根端鞭毛；
- 血琼脂上35℃培养18–24h：小、不透明、湿润、凸起菌落（部分产黄色色素）；
- 麦康凯琼脂上：小、湿润、半透明菌落；
- 生化反应：氧化酶(+)、触酶(+)、赖氨酸脱羧酶(+)，可分解麦芽糖和乳糖。

### 2. 分子确证（必须执行）
因表型鉴定不可靠，需采用以下至少一种方法：
- **recA基因特异性PCR**：区分Bcc内不同种；
- **多位点序列分型（MLST）**：用于流行病学追踪；
- **MALDI-TOF质谱**：若数据库包含Bcc参考谱。

### 3. 耐药性评估
- **天然耐药**：氨基糖苷类、多黏菌素；
- **获得性/诱导性耐药**：
  • 染色体编码的诱导型β-内酰胺酶 → 对青霉素类、头孢菌素类耐药；
  • 青霉素结合蛋白改变 → 降低β-内酰胺类亲和力。

### 4. 治疗与防控
- **首选抗生素**：复方磺胺甲唑、美罗培南或环丙沙星；
- **感染控制**：
  • 立即隔离患者（尤其囊性纤维化病房）；
  • 排查医院水源、消毒设备、药物制剂污染；
  • 启动院感暴发应急预案。

> **注意**：本技能仅适用于疑似Bcc菌株，不适用于其他伯克霍尔德菌属以外的细菌。
