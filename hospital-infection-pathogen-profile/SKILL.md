# 医院感染病原体构成特征分析

## 适用场景
- 临床医生评估疑似医院感染患者的病原学可能性
- 感染控制团队制定抗菌药物使用策略
- 微生物实验室优化检测流程

## 执行步骤
1. **确认感染类型**：仅适用于医院获得性感染，排除社区获得性感染。
2. **识别主要病原体类别**：
   - 细菌占比 >90%，以革兰氏阴性杆菌为主
   - 其他包括病毒、真菌（如白念珠菌）、原虫
3. **按感染部位匹配典型病原体**：
   - **泌尿道感染**：大肠埃希菌、肺炎克雷伯菌、鲍曼不动杆菌、表皮葡萄球菌等
   - **呼吸道感染**：流感嗜血杆菌、铜绿假单胞菌、鲍曼不动杆菌、金黄色葡萄球菌、呼吸道病毒等
   - **伤口/皮肤感染**：金黄色葡萄球菌、链球菌、厌氧菌、铜绿假单胞菌等
   - **胃肠道感染**：艰难拟梭菌、沙门菌、诺如病毒、轮状病毒、白念珠菌等
4. **考虑时代变迁趋势**：当前以革兰氏阴性杆菌和白念珠菌为主，区别于20世纪50–60年代以革兰氏阳性球菌为主的格局。

> **注意**：本技能不适用于非机会致病菌主导的感染或社区感染场景。
