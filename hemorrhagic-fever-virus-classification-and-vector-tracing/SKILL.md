# Hemorrhagic Fever Virus Classification and Vector Tracing

## 触发条件
当需对出血热病因进行分类、溯源或开展流行病学分析时使用本技能，尤其适用于发热伴出血症状的病例。

## 执行步骤

1. **确认共同临床特征**：
   - “3H”症状：高热（hyperpyrexia）、出血（hemorrhage）、低血压（hypotension）；
   - 病死率高，需快速识别病原。

2. **按病毒科属分类并匹配媒介与疾病**：
   - **汉坦病毒科**：汉坦病毒 → 啮齿动物（如黑线姬鼠）→ 肾综合征出血热、汉坦病毒肺综合征；
   - **内罗病毒科**：克里米亚-刚果出血热病毒 → 蜱 → 克里米亚-刚果出血热；
   - **黄病毒科**：登革病毒、黄热病病毒 → 蚊 → 登革出血热、黄热病；
   - **沙粒病毒科**：拉沙病毒等 → 啮齿动物 → 拉沙热等；
   - **丝状病毒科**：埃博拉病毒、马尔堡病毒 → 人传人为主，媒介未明 → 埃博拉出血热等。

3. **结合地理分布缩小范围**：
   - 中国常见：汉坦病毒、登革病毒、克里米亚-刚果出血热病毒；
   - 非洲：埃博拉、马尔堡、拉沙病毒；
   - 全球分布：部分黄病毒、汉坦病毒。

4. **排除非病毒性出血性疾病**。

5. **输出结果**：
   返回病毒科属、主要传播媒介、所致疾病名称及典型地理分布，用于病因鉴别与防控决策。
