## プロジェクト概要

このプロジェクトは、細菌学、ウイルス学、真菌学、免疫学、消毒・滅菌、微生物叢学、感染制御などの主要分野における **153 の重要なスキル**を、9 つのカテゴリーに体系的に統合しています。

**対象読者**: 医学生、臨床医、臨床検査技師、公衆衛生従事者、感染制御チーム

**参考教科書**: 人民衛生出版社『医学微生物学』第10版

**⚠️リスク⚠️**: このスキルセットには微生物学的診断、抗菌療法、感染制御、実験室手順が含まれており、独立した医療アドバイスとして誤用される可能性があります。
