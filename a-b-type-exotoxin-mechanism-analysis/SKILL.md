# A-B型外毒素结构与功能机制分析

## 何时使用
- 需要解析革兰氏阳性菌或阴性菌分泌的A-B型外毒素致病机制时
- 评估宿主细胞是否具备毒素作用条件（如受体表达）
- 设计类毒素疫苗或理解现有疫苗（如白喉/破伤风类毒素）原理时
- 判断毒素基因来源（染色体、质粒或前噬菌体）对毒力调控的影响

## 执行步骤

### 1. 确认毒素结构类型
- 判断A-B型外毒素是单一多肽链（如铜绿假单胞菌外毒素A）还是由独立A、B亚基通过二硫键连接（如霍乱毒素）。

### 2. 验证作用前提条件
- 确认细菌已分泌完整A-B结构外毒素。
- 确认宿主靶细胞表面表达B亚基特异性受体（如GM1-神经节苷脂）。
- 若任一亚基缺失或受体不表达，则毒素无法发挥毒性。

### 3. 分析内化与毒性过程
- B亚基结合宿主细胞受体 → 毒素-受体复合物内吞入胞 → A亚基在胞内释放 → A亚基发挥酶活性或毒性效应（如抑制蛋白合成、阻断神经递质释放）。

### 4. 确定毒素基因定位
- 查询毒素基因位于：
  - 染色体（如鲍特菌腺苷酸环化酶毒素）
  - 质粒（如破伤风痉挛毒素、炭疽毒素）
  - 前噬菌体（如霍乱毒素、白喉毒素、肉毒毒素）

### 5. 评估疫苗开发潜力
- B亚基通常为保护性抗原，可用于疫苗设计。
- 类毒素制备：用0.4%甲醛处理完整外毒素，去除毒性但保留免疫原性。

> **注意**：A或B亚基单独存在时无毒性；分子完整性是致病必要条件。
