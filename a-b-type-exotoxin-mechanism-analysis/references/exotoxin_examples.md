# 典型A-B型外毒素示例

## 白喉毒素（Diphtheria toxin）
- **来源**：携带tox基因的β-棒状杆菌噬菌体（前噬菌体）
- **受体**：肝素结合表皮生长因子样生长因子（HB-EGF）
- **A亚基作用**：ADP-核糖基化延伸因子2（EF-2），抑制真核细胞蛋白合成
- **临床表现**：心肌炎、神经麻痹

## 霍乱毒素（Cholera toxin）
- **来源**：CTXφ噬菌体（前噬菌体）
- **受体**：GM1-神经节苷脂
- **A亚基作用**：ADP-核糖基化Gsα蛋白，持续激活腺苷酸环化酶 → cAMP↑ → Cl⁻大量分泌 → 水样腹泻

## 肉毒毒素（Botulinum toxin）
- **来源**：肉毒梭菌质粒或染色体
- **受体**：突触前膜神经节苷脂及蛋白受体
- **A亚基作用**：锌依赖性蛋白酶，切割SNARE蛋白 → 阻断乙酰胆碱释放 → 弛缓性麻痹

## 炭疽毒素（Anthrax toxin）
- **组成**：保护性抗原（PA，B亚基）、致死因子（LF）、水肿因子（EF）
- **来源**：pXO1质粒
- **机制**：PA结合受体并形成通道，LF/EF进入细胞 → LF为金属蛋白酶破坏MAPKK通路，EF为腺苷酸环化酶

## 破伤风痉挛毒素（Tetanus toxin）
- **来源**：质粒编码
- **靶点**：脊髓抑制性中间神经元
- **作用**：切割VAMP/synaptobrevin → 阻断甘氨酸/GABA释放 → 肌肉强直痉挛
