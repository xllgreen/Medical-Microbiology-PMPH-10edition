# 细菌营养物质摄取机制详解

## 1. 被动扩散（Passive Diffusion）

### 简单扩散
- 物质直接穿过脂质双层膜
- 适用于小分子、非极性或脂溶性物质（如O₂、CO₂、乙醇）
- 速率与浓度梯度成正比

### 易化扩散（Facilitated Diffusion）
- 需特异性通道蛋白或载体蛋白
- 示例：甘油通过甘油通道蛋白进入细胞后，被甘油激酶磷酸化为3-磷酸甘油，从而在胞内积累
- 不消耗ATP，但依赖底物特异性蛋白

## 2. 主动转运（Active Transport）

### ABC转运系统（ATP-binding cassette transport）
- **革兰氏阴性菌**：特异性结合蛋白位于周浆间隙
- **革兰氏阳性菌**：结合蛋白锚定于细胞膜外表面
- 过程：
  1. 营养物 + 结合蛋白 → 复合物
  2. 复合物与膜上ABC载体结合
  3. ATP水解 → 构象变化 → 打开通道 → 物质入胞
- 常见底物：糖类、氨基酸、金属离子

### 离子偶联转运（Ion-coupled transport）
- 利用质子动力（PMF）或钠动力（SMF）
- 载体为疏水性膜蛋白（透性酶）
- 氧化态结合底物，还原态释放
- 在需氧呼吸细菌中占主导
- 示例：乳糖透过酶（LacY）利用H⁺梯度共转运乳糖

## 3. 基团转移（Group Translocation）

### 磷酸转移酶系统（PTS）
- 主要用于葡萄糖、甘露糖、果糖等糖类
- 能量供体：磷酸烯醇丙酮酸（PEP）
- 关键蛋白：EI、HPr、EIIA、EIIB、EIIC
- 步骤：
  1. PEP → EI → HPr（磷酸化）
  2. HPr~P → EIIA → EIIB
  3. EIIB~P 在膜外结合葡萄糖
  4. 葡萄糖经EIIC通道入胞，同时被EIIB磷酸化为Glc-6-P
- 优势：磷酸化产物无法逸出，实现高效积累

## 4. 机制选择影响因素
- **底物性质**：极性、分子大小、电荷
- **细菌代谢状态**：有氧/无氧、生长阶段
- **环境条件**：pH、离子强度、底物浓度
- **基因表达调控**：如大肠杆菌在葡萄糖存在时抑制其他糖转运系统（分解代谢物阻遏）
