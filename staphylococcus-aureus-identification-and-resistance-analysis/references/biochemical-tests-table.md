## 表9-1：金黄色葡萄球菌关键生化与表型特征

| 特征 | 结果 | 说明 |
|------|------|------|
| 血浆凝固酶 | 阳性 | 区分致病性金黄色葡萄球菌与其他葡萄球菌的关键 |
| β溶血 | 阳性 | 血琼脂上完全透明溶血环 |
| 甘露醇发酵 | 阳性 | 产酸，常用于甘露醇盐琼脂筛选 |
| 耐热核酸酶 | 阳性 | 热处理后仍具活性，特异性高 |
| SPA蛋白 | 阳性（>90%） | 可用于协同凝集试验 |
| 新生霉素敏感 | 敏感 | 区别于某些凝固酶阴性葡萄球菌 |
| 磷壁酸类型 | 核糖醇型 | 表皮葡萄球菌为甘油型 |

> 注：所有测试应在标准条件下进行，避免假阴性。
