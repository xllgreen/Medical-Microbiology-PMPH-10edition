# 郭霍法则病原体验证

## 何时使用
- 需确定某种微生物是否为特定感染性疾病的病原体
- 研究对象为细菌或可人工培养的病原体
- 具备实验动物模型和微生物纯培养条件
- 不适用于无法培养的病原体（如麻风分枝杆菌）、无动物模型的情况或病毒（需修正应用）

## 执行步骤

1. **病原体一致性检查**：确认该微生物在所有患病个体中均可检出，而在健康个体中不存在或极为罕见。
2. **分离与纯培养**：从患者样本中分离该微生物，并在人工培养基上获得纯培养物。
3. **动物致病性验证**：将纯培养物接种至易感实验动物，观察是否出现与人类患者相同的临床症状和病理变化。
4. **再分离确认**：从发病的实验动物体内再次分离该微生物，并验证其与原始病原体一致。

## 特殊情况处理
- 若病原体无法体外培养，可结合分子生物学（如PCR）或免疫学方法（如特异性抗体检测）作为替代证据。
- 对于无症状携带者，第1条应修正为“主要在患者中检出，健康人群中罕见或不存在”。
- 若缺乏合适动物模型，应整合流行病学、基因组学和临床数据进行综合判断。

## 输出结论
若四条均满足，则确认该微生物为该病的病原体；若部分不满足，需结合现代技术补充证据并谨慎得出结论。
