# 钩端螺旋体病显微镜凝集试验（MAT）诊断

## 适用场景
- 患者有钩端螺旋体病流行病学史和/或典型临床表现
- 已采集急性期和/或恢复期血清标本
- 发病时间超过1周（因抗体产生需1-2周）

## 执行步骤

1. **抗原准备**：使用我国15群15型致病性钩端螺旋体参考标准株，并结合当地常见血清群、型的活钩端螺旋体作为抗原。

2. **血清稀释与反应**：将患者血清进行系列稀释，与抗原混合后于37℃孵育1～2小时。

3. **显微镜观察**：在暗视野显微镜下检查凝集现象。阳性表现为钩端螺旋体凝集成不规则团块状或蜘蛛状；无凝集则为阴性。

4. **效价判定**：以50%钩端螺旋体被凝集的最高血清稀释度作为终点效价。

5. **诊断判读**：
   - 若单份血清MAT效价 ≥ 1:400 → 判定为阳性（支持钩端螺旋体病诊断）
   - 若双份血清（间隔2-3周）效价增长 ≥ 4倍 → 判定为确诊感染

6. **注意事项**：
   - 该试验特异性和敏感性高，但不适用于疾病早期（<1周）诊断
   - 抗原选择必须覆盖当地流行血清型，否则可能导致假阴性

## 输出结果
返回明确诊断结论：'阳性（支持钩端螺旋体病）' 或 '阴性（不支持）'，并附关键判读依据（如效价值、是否双份血清等）。
