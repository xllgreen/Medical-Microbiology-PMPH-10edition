# 大肠埃希菌致病型别鉴定与临床意义评估

## 触发条件
- 已确认分离株为大肠埃希菌
- 患者出现腹泻（水样泻、脓血便等）或肠外感染症状
- 需进一步分型以明确致病机制和临床管理策略

## 执行步骤

1. **收集临床与实验室信息**：
   - 症状类型（水样泻、血便、里急后重、持续性腹泻等）
   - 感染部位（小肠或大肠）
   - 流行病学史（旅行、食物暴露等）

2. **根据致病特征匹配型别**：
   - **肠产毒素型（ETEC）**：婴幼儿或旅行者水样泻；由LT/ST肠毒素介导；侵犯小肠。
   - **肠侵袭型（EIEC）**：脓血便、里急后重；类似志贺菌；侵犯结肠；多见于较大儿童和成人。
   - **肠致病型（EPEC）**：婴幼儿腹泻为主；通过T3SS引起微绒毛破坏；无肠毒素。
   - **肠出血型（EHEC/STEC）**：血便，可进展为HUS（溶血性尿毒综合征）；产志贺毒素（Stx）；O157:H7常见；感染剂量极低。
   - **肠集聚型（EAEC）**：持续性水样泻，偶有血便；形成“砖状”生物被膜；多见于婴儿和旅行者。

3. **评估临床风险**：
   - 若为EHEC，立即评估HUS风险（监测肾功能、血小板、溶血指标）
   - 根据型别决定是否使用抗生素（如EHEC通常禁用）

4. **输出建议**：
   - 明确致病型别
   - 提示并发症预警（如HUS）
   - 建议隔离与公共卫生报告（尤其EHEC）

> 注意：本技能不适用于非致病性大肠埃希菌或非肠杆菌科细菌。
