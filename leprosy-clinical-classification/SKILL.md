# 麻风病临床分型

## 适用场景
当患者出现慢性皮肤或神经病变，且临床高度怀疑麻风病时，结合以下检查结果执行本分型流程：
- 临床症状观察
- 皮肤或神经病变活检
- 麻风菌素试验结果
- 涂片抗酸染色

## 分型步骤

1. **评估麻风菌素试验结果**：
   - 阳性 → 提示细胞免疫正常，倾向结核样型或未定类；
   - 阴性 → 提示T细胞免疫缺陷，倾向瘤型或界线类。

2. **评估细菌载量**（通过涂片抗酸染色或活检）：
   - 高载量（大量抗酸杆菌）→ 支持瘤型；
   - 极低或无抗酸杆菌 → 支持结核样型或未定类；
   - 中等载量 → 考虑界线类。

3. **结合临床特征判断具体分型**：
   - **瘤型麻风**：广泛皮肤/黏膜/内脏受累，“狮面容”，高传染性，麻风菌素阴性；
   - **结核样型麻风**：局限性皮损伴神经损害，自限倾向，麻风菌素阳性；
   - **界线类麻风**：兼具两型特征，可向任一极转化；
   - **未定类麻风**：早期轻微皮损，多转为结核样型，麻风菌素多阳性。

4. **输出结果**：明确分型名称，并附对应的免疫状态、传染性及预后。

## 注意事项
- 本分型不适用于其他慢性肉芽肿性皮肤病（如结核样肉芽肿）；
- 需排除非结核分枝杆菌感染；
- 免疫状态主要依赖细胞免疫，机制类似抗结核免疫。
