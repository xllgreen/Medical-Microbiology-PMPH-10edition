# 朊粒检测与强化消毒

## 触发条件
当出现以下任一情况时使用本技能：
- 临床怀疑朊粒病（如快速进展性痴呆、肌阵挛、共济失调）
- 需对脑脊液、脑组织或手术器械进行朊粒检测或去污染处理

## 执行前准备
确认满足以下前提：
- 已获取脑脊液或脑组织样本（用于检测）
- 或已识别需处理的潜在污染物品（如神经外科器械）
- 明确常规灭菌方法（乙醇、紫外线、标准高压灭菌）对朊粒无效

## 核心操作流程

### 1. PrP^Sc 检测方法
- **RT-QuIC（首选）**：将脑脊液与重组PrP^C和硫黄素T混合，经振荡-静置循环，若存在PrP^Sc则诱导聚集并发出荧光，实现高灵敏度、高特异性诊断
- **PMCA**：将样本与正常人脑匀浆混合，经超声-静置循环扩增PrP^Sc，产物用免疫印迹检测
- **免疫印迹**：脑组织匀浆经蛋白酶K处理后，用PrP抗体检测蛋白酶抗性条带
- **免疫组化（IHC）**：脑切片经高温+甲酸预处理后，用PrP抗体染色观察淀粉样沉积（注意：灵敏度较低）

### 2. 辅助生物标志物
- 检测脑脊液中14-3-3蛋白（ELISA或免疫印迹），仅作辅助参考（非特异性）

### 3. 遗传学分析
- 对PRNP基因进行PCR扩增与测序，确认是否存在致病突变（如D178N、P102L），用于家族性病例确诊

### 4. 强化消毒灭菌（WHO推荐）
- **耐热物品**：
  1. 浸泡于1 mol/L NaOH 或 2% 次氯酸钠 1小时
  2. 清水彻底冲洗
  3. 121°C（下排气）或134°C（预排气）压力蒸汽灭菌1小时
- **热敏感物品或表面**：
  - 用2 mol/L NaOH 或未稀释次氯酸钠冲洗，静置1小时
  - 擦干后清水冲洗
- **绝对禁止**：将朊粒病患者组织用于器官或组织移植

## 关键注意事项
- 朊粒对尿素、甲醛、乙醇、常规加热、紫外线及电离辐射具有高度抗性
- 所有操作需在专用区域进行，避免交叉污染
- 消毒后物品需验证去污染效果（如生物指示剂）
