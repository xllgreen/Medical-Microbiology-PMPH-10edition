# 肠杆菌科细菌微生物学检查

## 何时使用
- 临床怀疑肠杆菌科细菌引起的感染（如尿路感染、败血症、胃肠炎、脑膜炎等）
- 已获得符合无菌要求和时效标准的临床标本
- 实验室具备基础微生物培养与鉴定条件

## 执行步骤

### 1. 标本预处理
- **肠外感染标本**（血液、尿液、脑脊液）：先进行增菌培养，再转种鉴定
- **直接标本**（脓汁、痰）：先做革兰染色镜检，再划线接种于血琼脂平板
- **粪便标本**：直接接种于选择性鉴别培养基（如MAC、EMB或SS琼脂）

### 2. 分离培养与初步鉴定
- 在MAC/EMB/SS琼脂上观察菌落特征：
  - 乳糖发酵菌（如大肠埃希菌）→ 红色菌落（MAC）或紫黑色带金属光泽（EMB）
  - 非发酵菌（如志贺菌、沙门菌）→ 无色透明菌落
- 挑选可疑菌落接种至克氏双糖铁琼脂（KIA），观察产酸、产气、H₂S产生及动力等生化反应

### 3. 血清学分型（如需要）
- 使用特异性抗血清进行O、H、K抗原分型，用于流行病学调查或疑难菌株确认

### 4. 自动化与新技术辅助鉴定
- 可使用自动化生化鉴定系统进行快速鉴定
- MALDI-TOF MS可用于培养物快速鉴定，但对大肠埃希菌与志贺菌区分能力有限，建议结合AI增强软件提高准确性
