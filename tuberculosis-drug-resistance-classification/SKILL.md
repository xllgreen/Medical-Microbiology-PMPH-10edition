# 结核分枝杆菌耐药类型分类

## 适用条件
- 已完成一线抗结核药物药敏试验
- 明确异烟肼、利福平等关键药物的敏感性
- 仅适用于结核分枝杆菌复合群（不适用于非结核分枝杆菌）

## 分类步骤

1. **判断是否为单耐药结核病（MR-TB）**：
   - 若仅对一种一线抗结核药物耐药，则归类为 MR-TB。

2. **判断是否为多耐药结核病（PDR-TB）**：
   - 若对两种或以上一线抗结核药物耐药，但**未同时耐异烟肼和利福平**，则归类为 PDR-TB。

3. **判断是否为耐多药结核病（MDR-TB）**：
   - 若**同时对异烟肼和利福平耐药**，无论其他药物敏感性如何，均归类为 MDR-TB。

4. **判断是否为广泛耐药结核病（XDR-TB）**：
   - 在已确认为 MDR-TB 的前提下，若**对任一氟喹诺酮类药物耐药**或**对任一二线注射类药物耐药**（满足其一即可），则归类为 XDR-TB。

## 关键药物定义
- **一线抗结核药物**：异烟肼、利福霉素类（如利福平）、吡嗪酰胺、乙胺丁醇、链霉素
- **二线注射类药物**：阿米卡星、卡那霉素、卷曲霉素
- **氟喹诺酮类药物**：莫西沙星、左氧氟沙星等

## 输出要求
输出唯一耐药类型：`MR-TB`、`PDR-TB`、`MDR-TB` 或 `XDR-TB`，用于指导后续临床治疗决策。
