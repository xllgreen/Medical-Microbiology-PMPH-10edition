# 肺炎衣原体感染的血清学诊断

## 适用场景
- 患者疑似肺炎衣原体感染
- 已采集单份或双份（急性期与恢复期）血清标本

## 执行步骤
1. 使用微量免疫荧光试验（MIF）测定血清中特异性IgM和IgG抗体滴度。
2. 根据血清样本类型应用以下判断标准：
   - **双份血清**：若恢复期IgG抗体滴度较急性期增高≥4倍 → 判定为**急性感染**。
   - **单份血清**：
     - IgM滴度 ≥ 1:16 → 提示**近期/急性感染**；
     - IgG滴度 ≥ 1:512 → 提示**急性感染**；
     - IgG滴度 ≥ 1:16 但 < 1:512 → 判定为**既往感染**。
3. 综合判断感染状态，并注意该方法可区分原发与继发感染。

## 注意事项
- 感染早期抗体可能未产生，导致假阴性结果。
- 本标准仅适用于肺炎衣原体，不可直接用于其他衣原体属感染的诊断。
