# 细菌耐药表型四级分类

## 何时使用
- 已完成标准化药敏试验（依据CLSI或EUCAST折点）；
- 需对临床分离菌株（医院或社区来源）进行耐药程度分类；
- 用于指导抗菌药物选择、感染防控或公共卫生报告。

> **注意**：不适用于结核分枝杆菌（有特殊标准），且需排除技术误差导致的假耐药。

## 执行步骤

### 1. 单耐药性（SDR）
- **定义**：仅对一种抗菌药物耐药；
- **示例**：金黄色葡萄球菌对阿米卡星耐药。

### 2. 多重耐药性（MDR）
- **定义**：同时对**三类或以上**作用机制不同的抗菌药物耐药；
- **关键**：必须跨越不同药物类别（如β-内酰胺类 + 氨基糖苷类 + 大环内酯类）。

### 3. 广泛耐药性（XDR）
- **定义**：对几乎所有临床常用抗菌药物耐药，**仅保留1–2类敏感**；
  - **革兰氏阴性菌**：仅对多黏菌素和/或替加环素敏感；
  - **革兰氏阳性菌**：仅对糖肽类（如万古霉素）和/或利奈唑胺敏感。

### 4. 泛耐药性（PDR）
- **定义**：对**所有临床常用抗菌药物均耐药**；
  - **革兰氏阴性菌**：包括多黏菌素和替加环素在内的全部药物均耐药；
  - **革兰氏阳性菌**：包括糖肽类和利奈唑胺在内的全部药物均耐药。

> **说明**：“临床常用抗菌药物”指当前常规用于治疗的药物，不包括实验性或极少使用药物。

## 输出结果
输出耐药表型分类标签（SDR / MDR / XDR / PDR），用于指导治疗策略（如XDR/PDR需启用最后防线药物）和感染控制措施。
