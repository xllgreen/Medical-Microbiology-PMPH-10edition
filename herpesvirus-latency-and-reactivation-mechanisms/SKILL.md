# 疱疹病毒潜伏感染与再激活机制分析

## 何时使用
- 患者有明确疱疹病毒感染史，出现疑似复发症状
- 患者处于免疫抑制状态（如器官移植、HIV感染、长期使用激素）
- 需评估发热、日晒、情绪紧张等应激因素是否诱发病毒再激活
- 鉴别急性感染与潜伏再激活所致的临床表现

## 执行步骤

### 1. 确认潜伏建立条件
- 验证患者已完成原发疱疹病毒感染
- 确认宿主已产生特异性免疫应答
- 判断病毒是否已进入特定潜伏部位：
  - HSV-1：三叉神经节、颈上神经节
  - HSV-2：骶神经节
  - VZV：脊髓后根神经节
  - HCMV/HHV-6/7：髓系前体细胞、唾液腺、淋巴细胞
  - EBV/HHV-8：B细胞

### 2. 识别再激活诱因
- **非特异性因素**：发热、寒冷、紫外线暴露、月经期、情绪应激、合并其他感染
- **免疫抑制状态**：器官移植、HIV/AIDS、长期糖皮质激素或免疫抑制剂使用、恶性肿瘤

### 3. 分析再激活分子过程
- 即刻早期蛋白（α蛋白）使环状病毒基因组线性化
- 启动级联表达：早期蛋白（β蛋白）→晚期蛋白（γ蛋白）
- 病毒在细胞核内复制DNA并组装核衣壳
- 通过出芽获得包膜，以胞吐或细胞溶解方式释放，或经细胞间桥直接扩散

### 4. 评估临床后果
- 判断是否为同一部位复发性局部疱疹
- 注意病程通常较原发感染轻、短，但仍具传染性
- 警惕重症并发症：脑炎、播散性感染（如新生儿HSV感染病死率高达80%）

## 输出建议
提示临床医生关注患者当前免疫状态及近期应激暴露史，在高危人群中主动筛查疱疹病毒再激活迹象，并考虑预防性抗病毒治疗。
