# 干扰素抗病毒作用机制

## 何时使用
- 分析病毒感染后宿主细胞如何建立抗病毒状态
- 评估干扰素类药物在特定病毒治疗中的适用性
- 解释为何干扰素对某些细胞或病毒无效（如缺乏受体、种属不匹配）
- 理解病毒逃逸干扰素作用的可能机制

## 执行步骤
1. **确认前提条件**：
   - 存在干扰素诱生剂（如病毒、细菌内毒素、双链RNA）
   - 靶细胞表面表达相应干扰素受体
   - 病毒复制过程中产生双链RNA（dsRNA）

2. **识别干扰素类型与来源**：
   - Ⅰ型（IFN-α、IFN-β）、Ⅱ型（IFN-γ）或Ⅲ型（IFN-λ）由巨噬细胞、淋巴细胞、成纤维细胞等分泌

3. **追踪信号传导路径**：
   - 干扰素结合邻近细胞表面受体，激活JAK-STAT通路，诱导抗病毒蛋白（AVP）基因转录

4. **分析两条核心抗病毒通路**：
   - **2'-5'A合成酶通路**：
     - dsRNA激活2'-5'A合成酶 → 生成2'-5'寡聚腺苷酸（2'-5'A）→ 激活RNaseL → 降解病毒mRNA
   - **PKR通路**：
     - dsRNA激活蛋白激酶R（PKR）→ PKR磷酸化eIF2-α → 抑制病毒蛋白翻译起始

5. **评估作用特征**：
   - 不直接杀灭病毒，仅抑制其复制与扩散
   - 起效快（数小时内），持续2–3天
   - 具有种属特异性（同种细胞间有效）

6. **判断失效情形**：
   - 细胞缺乏干扰素受体
   - 病毒已进化出逃逸机制（如编码dsRNA屏蔽蛋白）
   - 跨种属应用（如人干扰素用于动物细胞）
