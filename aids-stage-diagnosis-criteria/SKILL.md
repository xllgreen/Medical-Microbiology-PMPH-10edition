# 艾滋病期诊断标准应用

## 诊断条件（满足任一即成立）
- **免疫学标准**：CD4⁺T细胞 < 200 个/μL（无论有无症状）
- **临床标准**：出现任一AIDS指征性疾病（即使CD4≥200）

## 执行步骤
1. **确认HIV感染已确诊**（抗体/抗原/核酸检测阳性）
2. **筛查指征性疾病**：
   - **真菌**：肺孢子菌肺炎（PCP）、隐球菌病、念珠菌病
   - **细菌**：播散性结核、李斯特菌病
   - **病毒**：CMV视网膜炎、HSV/VZV严重感染
   - **原虫**：隐孢子虫腹泻、弓形体脑炎
   - **肿瘤**：卡波西肉瘤、Burkitt淋巴瘤、HPV相关癌
3. **立即行动**：
   - 启动抗逆转录病毒治疗（ART）
   - 针对并发症进行特异性治疗

> **注意**：需排除其他免疫抑制病因；不适用于HIV急性期或无症状期。
